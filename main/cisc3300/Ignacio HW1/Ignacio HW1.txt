1. What happens from the time a user puts a URL into their browser to when they are able to view a webpage? - Briefly define ISP, URL, DNS, web server.

	First, the user has to be connected to an internet service provider (ISP) which is in charge of connecting their web browsers to the internet. The user then has to type in a the uniform resource locator (URL) of the page they want to visit into their web browser. URL’s are webpages’ unique addresses. The computer then will utilize domain name system (DNS) servers to connect the user to the corresponding IP address of the webpage they wish to visit based on the URL they input. This then leads the users’ local computer to connect to web servers that are in charge of continuously running webpages users want to visit.

3. Access https://www.fordham.edu/. In your own words, describe how this webpage uses structure to convey meaning and navigation options to the user.

	The Fordham homepage uses a combination of headings, images, icons, and menus to organize the main landing page. Different headings and body texts are used to relay the hierarchy of the information. For instance, texts in bold represent headlines and section titles while smaller text is used for descriptions and captions. Furthermore, images are also used to further signify the different sections and headlines. Other structural elements like clickable icons and a slide-in menu are also used for easy navigation and organization throughout the page.

4. What is an html element? Tag? Attribute?

	HTML elements describe the structure and organization of webpages. HTML tags act like labels for the information that is put in between them. HTML attributes contain additional information specific to the elements.
